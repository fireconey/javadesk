闭包的目的是不能使外部直接调用某个函数。闭包可以注入语法，同时传入此语法的参数。

def g(fun):   #第一个可以注入任何语法
     def h(x):  #这个可以注入对应语法的参数
           fun(x)
    return h


Python把一切都看为对象，函数也是对象，所以可以像类一样在内部定义其他函数
如果return的函数没有括号，说明返回对象，对象是没有运行的图纸，是这个对象的整体结构，所以不要传入参数，如果有括号就是返回函数，是返回的这个函数运行的结果，则这个函数有参数必须传入参数

def ji():
       def ko(x):
            return 1
      return ko  #这里返回的是对象不要参数（返回的是def ko（x）这个结构，不是计算后的值），如果有括号则必须有给定具体的参数

def ji():
     def ko():
           return 1
     return ko

print(ji()())#第一个括号表示返回是ji函数不是def ji():def ko（）整个包（对象）。第二个括号
由于返回的是ko对象所以要解析为函数。
